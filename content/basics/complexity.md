## –í—ã—á–∏—Å–ª–µ–Ω–∏–µ —á–∏—Å–µ–ª –§–∏–±–æ–Ω–∞—á—á–∏

–ü–æ—Å–ª–µ–¥–æ–≤–∞—Ç–µ–ª—å–Ω–æ—Å—Ç—å —á–∏—Å–µ–ª –§–∏–±–æ–Ω–∞—á—á–∏

$$

$$

–æ–ø—Ä–µ–¥–µ–ª—è–µ—Ç—Å—è —Å–ª–µ–¥—É—é—â–∏–º –æ–±—Ä–∞–∑–æ–º:


>[!def] –û–ø—Ä–µ–¥–µ–ª–µ–Ω–∏–µ –ø–æ—Å–ª–µ–¥–æ–≤–∞—Ç–µ–ª—å–Ω–æ—Å—Ç–∏ —á–∏—Å–µ–ª –§–∏–±–æ–Ω–∞—á—á–∏
>$F_{n}$ - $n$-–æ–µ —á–∏—Å–ª–æ –§–∏–±–æ–Ω–∞—á—á–∏, –≥–¥–µ $F_{0}=F_{1}=1, F_{n}=F_{n-1}+F_{n-2}, n>1 .$

–ß–∏—Å–ª–∞ –§–∏–±–æ–Ω–∞—á—á–∏ —Ä–∞—Å—Ç—É—Ç —ç–∫—Å–ø–æ–Ω–µ–Ω—Ü–∏–∞–ª—å–Ω–æ –±—ã—Å—Ç—Ä–æ:

>[!lemma] **–õ–µ–º–º–∞**
>$2^{\lfloor n / 2\rfloor} \leqslant F_{n} \leqslant 2^{n}$.
>{{% notice style="prove" expanded="true" %}}
–î–æ–∫–∞–∂–µ–º —É—Ç–≤–µ—Ä–∂–¥–µ–Ω–∏–µ –ø–æ –∏–Ω–¥—É–∫—Ü–∏–∏.

–ë–∞–∑–∞. –£—Ç–≤–µ—Ä–∂–¥–µ–Ω–∏–µ –≤–µ—Ä–Ω–æ –¥–ª—è $n=0,1$.

–ü–µ—Ä–µ—Ö–æ–¥. –ü—É—Å—Ç—å $n>1$, —Ç–æ–≥–¥–∞ $F_{n}=F_{n-1}+F_{n-2} \leqslant 2^{n-1}+2^{n-2}<2^{n}$.

–° –¥—Ä—É–≥–æ–π —Å—Ç–æ—Ä–æ–Ω—ã, $F_{n} \geqslant 2 \cdot F_{n-2} \geqslant 2 \cdot 2^{\lfloor(n-2) / 2\rfloor}=2^{\lfloor n / 2\rfloor}$.
{{% /notice %}}

–ù–µ—Å–ª–æ–∂–Ω–æ –¥–æ–∫–∞–∑–∞—Ç—å –ø–æ –∏–Ω–¥—É–∫—Ü–∏–∏, —á—Ç–æ $F_{n} \leqslant T(n)<2 F_{n}$. –í –∫–∞–∂–¥–æ–º –≤—ã–∑–æ–≤–µ fib —Å–æ–≤–µ—Ä—à–∞–µ—Ç—Å—è –æ–≥—Ä–∞–Ω–∏—á–µ–Ω–Ω–æ–µ —á–∏—Å–ª–æ (—Å–∫–∞–∂–µ–º, –Ω–µ –±–æ–ª—å—à–µ –ø—è—Ç–∏) –æ–ø–µ—Ä–∞—Ü–∏–π, –ø–æ—ç—Ç–æ–º—É –≤—Ä–µ–º—è —Ä–∞–±–æ—Ç—ã –∞–ª–≥–æ—Ä–∏—Ç–º–∞ –ø—Ä–∏–º–µ—Ä–Ω–æ –ø—Ä–æ–ø–æ—Ä—Ü–∏–æ–Ω–∞–ª—å–Ω–æ $F_{n}$ (—á—É—Ç—å –ø–æ–∑–∂–µ —É –Ω–∞—Å –ø–æ—è–≤–∏—Ç—Å—è —Ñ–æ—Ä–º–∞–ª—å–Ω–æ–µ –æ–ø—Ä–µ–¥–µ–ª–µ–Ω–∏–µ —ç—Ç–æ–≥–æ "–ø—Ä–∏–º–µ—Ä–Ω–æ").

–ò–∑ –ª–µ–º–º—ã —Å–ª–µ–¥—É–µ—Ç, –Ω–∞–ø—Ä–∏–º–µ—Ä, —á—Ç–æ, $F_{300} \geqslant 2^{150}>10^{45}$. –ù–∞ –∫–æ–º–ø—å—é—Ç–µ—Ä–µ, –≤—ã–ø–æ–ª–Ω—è—é—â–µ–º $10^{9}$ –æ–ø–µ—Ä–∞—Ü–∏–π –≤ —Å–µ–∫—É–Ω–¥—É, fib(300) –±—É–¥–µ—Ç –≤—ã–ø–æ–ª–Ω—è—Ç—å—Å—è –±–æ–ª—å—à–µ $10^{36}$ —Å–µ–∫—É–Ω–¥.

```py
def fibFast(n: int) -> int:
    """
    Calculate nth Fibonacci number using dynamic programming.
    
    Args:
        n: Non-negative integer index
        
    Returns:
        nth Fibonacci number (F(0)=1, F(1)=1, F(2)=2, ...)
    """
    if n <= 1:
        return 1
    
    # Create array with indices 0..n
    f = [0] * (n + 1)
    f[0] = f[1] = 1
    
    for i in range(2, n + 1):
        f[i] = f[i - 1] + f[i - 2]
    
    return f[n]
```

–ù–∞ –∫–∞–∂–¥–æ–π –∏—Ç–µ—Ä–∞—Ü–∏–∏ —Ü–∏–∫–ª–∞ —Å–æ–≤–µ—Ä—à–∞–µ—Ç—Å—è –æ–¥–Ω–æ —Å–ª–æ–∂–µ–Ω–∏–µ, –≤—Å–µ–≥–æ –∏—Ç–µ—Ä–∞—Ü–∏–π –ø—Ä–∏–º–µ—Ä–Ω–æ $n$, –ø–æ—ç—Ç–æ–º—É –∫–æ–ª–∏—á–µ—Å—Ç–≤–æ —Å–ª–æ–∂–µ–Ω–∏–π, –≤—ã–ø–æ–ª–Ω—è–µ–º—ã—Ö –≤ —Ö–æ–¥–µ –Ω–æ–≤–æ–≥–æ –∞–ª–≥–æ—Ä–∏—Ç–º–∞, –ø—Ä–∏–º–µ—Ä–Ω–æ –ø—Ä–æ–ø–æ—Ä—Ü–∏–æ–Ω–∞–ª—å–Ω–æ $n$. –ï—Å—Ç—å –µ—â—ë –æ–¥–Ω–∞ —Ç–æ–Ω–∫–æ—Å—Ç—å - –∏–∑ –ª–µ–º–º—ã —Å–ª–µ–¥—É–µ—Ç, —á—Ç–æ –¥–≤–æ–∏—á–Ω–∞—è –∑–∞–ø–∏—Å—å $F_{n}$ –∏–º–µ–µ—Ç –¥–ª–∏–Ω—É –ø–æ—Ä—è–¥–∫–∞ $n$. –ß—É—Ç—å –ø–æ–∑–∂–µ –º—ã —É–≤–∏–¥–∏–º, —á—Ç–æ —Å–ª–æ–∂–µ–Ω–∏–µ –¥–≤—É—Ö $n$-–±–∏—Ç–æ–≤—ã—Ö —á–∏—Å–µ–ª —Ç—Ä–µ–±—É–µ—Ç –ø–æ—Ä—è–¥–∫–∞ $n$ —ç–ª–µ–º–µ–Ω—Ç–∞—Ä–Ω—ã—Ö –æ–ø–µ—Ä–∞—Ü–∏–π, –∑–Ω–∞—á–∏—Ç –æ–±—â–µ–µ –≤—Ä–µ–º—è —Ä–∞–±–æ—Ç—ã –Ω–æ–≤–æ–≥–æ –∞–ª–≥–æ—Ä–∏—Ç–º–∞ –ø—Ä–∏–º–µ—Ä–Ω–æ –ø—Ä–æ–ø–æ—Ä—Ü–∏–æ–Ω–∞–ª—å–Ω–æ $n^{2}$. –° –ø–æ–º–æ—â—å—é _fibFast_ –º–æ–∂–Ω–æ —É–∂–µ –∑–∞ —Ä–∞–∑—É–º–Ω–æ–µ –≤—Ä–µ–º—è –≤—ã—á–∏—Å–ª–∏—Ç—å –Ω–µ —Ç–æ–ª—å–∫–æ $F_{300}$, –Ω–æ –∏ $F_{100000}$.

### –û-—Å–∏–º–≤–æ–ª–∏–∫–∞

–•–æ—Ä–æ—à–æ —Å–µ–±—è –∑–∞—Ä–µ–∫–æ–º–µ–Ω–¥–æ–≤–∞–ª –∏ —Å—Ç–∞–ª –æ–±—â–µ–ø—Ä–∏–Ω—è—Ç—ã–º —Å–ª–µ–¥—É—é—â–∏–π –ø–æ–¥—Ö–æ–¥ - –æ—Ü–µ–Ω–∏–≤–∞—Ç—å –≤—Ä–µ–º—è —Ä–∞–±–æ—Ç—ã –∞–ª–≥–æ—Ä–∏—Ç–º–∞ –Ω–µ–∫–æ—Ç–æ—Ä–æ–π —Ñ—É–Ω–∫—Ü–∏–µ–π –æ—Ç –≤—Ö–æ–¥–Ω—ã—Ö –ø–∞—Ä–∞–º–µ—Ç—Ä–æ–≤, –ø—Ä–∏ —ç—Ç–æ–º –ø—Ä–µ–Ω–µ–±—Ä–µ–≥–∞—è –æ–≥—Ä–∞–Ω–∏—á–µ–Ω–Ω—ã–º–∏ –º–Ω–æ–∂–∏—Ç–µ–ª—è–º–∏. –≠—Ç–æ –ø–æ–∑–≤–æ–ª—è–µ—Ç —ç—Ñ—Ñ–µ–∫—Ç–∏–≤–Ω–æ —Å—Ä–∞–≤–Ω–∏–≤–∞—Ç—å –∞–ª–≥–æ—Ä–∏—Ç–º—ã –º–µ–∂–¥—É —Å–æ–±–æ–π, –ø—Ä–∏ —ç—Ç–æ–º –Ω–µ –Ω—É–∂–Ω–æ –∑–∞–Ω–∏–º–∞—Ç—å—Å—è —Ç–æ—á–Ω—ã–º –ø–æ–¥—Å—á—ë—Ç–æ–º –∫–æ–ª–∏—á–µ—Å—Ç–≤–∞ —ç–ª–µ–º–µ–Ω—Ç–∞—Ä–Ω—ã—Ö –æ–ø–µ—Ä–∞—Ü–∏–π. –ü—Ä–∏–º–µ—Ä–Ω–æ —Ç–∞–∫ –º—ã –∏ —Ä–∞—Å—Å—É–∂–¥–∞–ª–∏ –æ–± –∞–ª–≥–æ—Ä–∏—Ç–º–∞—Ö –≤—ã—á–∏—Å–ª–µ–Ω–∏—è —á–∏—Å–µ–ª –§–∏–±–æ–Ω–∞—á—á–∏. –í–≤–µ–¥—ë–º –Ω–µ—Å–∫–æ–ª—å–∫–æ –æ–±–æ–∑–Ω–∞—á–µ–Ω–∏–π, –∫–æ—Ç–æ—Ä—ã–µ –ø–æ–º–æ–≥—É—Ç –ø—Ä–æ–≤–æ–¥–∏—Ç—å –ø–æ–¥–æ–±–Ω—ã–µ —Ä–∞—Å—Å—É–∂–¥–µ–Ω–∏—è –±–æ–ª–µ–µ –∫—Ä–∞—Ç–∫–æ –∏ —Ç–æ—á–Ω–æ.

‚úçÔ∏è –í—Ä–µ–º—è —Ä–∞–±–æ—Ç—ã –∞–ª–≥–æ—Ä–∏—Ç–º–∞ - —ç—Ç–æ, –∫–æ–Ω–µ—á–Ω–æ, –≤—Å–µ–≥–¥–∞ –Ω–µ–æ—Ç—Ä–∏—Ü–∞—Ç–µ–ª—å–Ω–∞—è —Ñ—É–Ω–∫—Ü–∏—è. –¢–µ–º –Ω–µ –º–µ–Ω–µ–µ, –º—ã –¥–∞—ë–º —Å–ª–µ–¥—É—é—â–∏–µ –æ–ø—Ä–µ–¥–µ–ª–µ–Ω–∏—è –¥–ª—è —Ñ—É–Ω–∫—Ü–∏–π, –ø—Ä–∏–Ω–∏–º–∞—é—â–∏—Ö –ø—Ä–æ–∏–∑–≤–æ–ª—å–Ω—ã–µ –≤–µ—â–µ—Å—Ç–≤–µ–Ω–Ω—ã–µ –∑–Ω–∞—á–µ–Ω–∏—è, –ø–æ—Å–∫–æ–ª—å–∫—É —Ç–∞–∫–∏–µ —Ñ—É–Ω–∫—Ü–∏–∏ —á–∞—Å—Ç–æ –≤–æ–∑–Ω–∏–∫–∞—é—Ç –≤ –ø—Ä–æ—Ü–µ—Å—Å–µ —Ä–∞—Å—Å—É–∂–¥–µ–Ω–∏–π.

{{% notice style="def" title="–û–ø—Ä–µ–¥–µ–ª–µ–Ω–∏–µ $O$, $\Omega$, $\Theta$" %}}
–†–∞—Å—Å–º–æ—Ç—Ä–∏–º —Ñ—É–Ω–∫—Ü–∏–∏ $f, g: \mathbb{N} \rightarrow \mathbb{R}$.
- $f=O(g)$, –µ—Å–ª–∏ —Å—É—â–µ—Å—Ç–≤—É—é—Ç —Ç–∞–∫–∏–µ $C>0, N>0$, —á—Ç–æ –¥–ª—è –ª—é–±–æ–≥–æ $n>N$ : $|f(n)| \leqslant C \cdot|g(n)|$.
- $f=\Omega(g)$, –µ—Å–ª–∏ —Å—É—â–µ—Å—Ç–≤—É—é—Ç —Ç–∞–∫–∏–µ $C>0, N>0$, —á—Ç–æ –¥–ª—è –ª—é–±–æ–≥–æ $n>N$ : $|f(n)| \geqslant C \cdot|g(n)|$.
- $f=\Theta(g)$, –µ—Å–ª–∏ —Å—É—â–µ—Å—Ç–≤—É—é—Ç —Ç–∞–∫–∏–µ $C_{1}>0, C_{2}>0, N>0$, —á—Ç–æ –¥–ª—è –ª—é–±–æ–≥–æ $n>N$ : $C_{1} \cdot|g(n)| \leqslant|f(n)| \leqslant C_{2} \cdot|g(n)|$.
{{% /notice %}}

‚úçÔ∏è –ó–∞–ø–∏—Å—å $f=O(g)$ –º–æ–∂–Ω–æ –ø–æ–Ω–∏–º–∞—Ç—å –∫–∞–∫ " $|f| \leqslant|g|$ —Å —Ç–æ—á–Ω–æ—Å—Ç—å—é –¥–æ –∫–æ–Ω—Å—Ç–∞–Ω—Ç—ã". –ê–Ω–∞–ª–æ–≥–∏—á–Ω–æ, $\Omega(\cdot)$ –º–æ–∂–Ω–æ —Å—á–∏—Ç–∞—Ç—å –∞–Ω–∞–ª–æ–≥–æ–º $\geqslant$, –∞ $\Theta(\cdot)$ - –∞–Ω–∞–ª–æ–≥–æ–º $=$.
–ï—â—ë –æ–¥–∏–Ω —Å–ø–æ—Å–æ–± –ø–æ–Ω–∏–º–∞—Ç—å –∑–∞–ø–∏—Å—å $f=O(g)$ - –æ—Ç–Ω–æ—à–µ–Ω–∏–µ $\frac{|f(n)|}{|g(n)|}$ –æ–≥—Ä–∞–Ω–∏—á–µ–Ω–æ —Å–≤–µ—Ä—Ö—É –Ω–µ–∫–æ—Ç–æ—Ä–æ–π –∫–æ–Ω—Å—Ç–∞–Ω—Ç–æ–π.

–ú–Ω–æ–≥–∏–µ –µ—Å—Ç–µ—Å—Ç–≤–µ–Ω–Ω—ã–µ —Å–≤–æ–π—Å—Ç–≤–∞ –æ–ø–µ—Ä–∞—Ç–æ—Ä–æ–≤ —Å—Ä–∞–≤–Ω–µ–Ω–∏—è $\leqslant,=, \geqslant$ –≤—ã–ø–æ–ª–Ω—è—é—Ç—Å—è –∏ –¥–ª—è –∏—Ö –∞—Å–∏–º–ø—Ç–æ—Ç–∏—á–µ—Å–∫–∏—Ö –∞–Ω–∞–ª–æ–≥–æ–≤. –°—Ñ–æ—Ä–º—É–ª–∏—Ä—É–µ–º –Ω–µ–∫–æ—Ç–æ—Ä—ã–µ –∏–∑ —ç—Ç–∏—Ö —Å–≤–æ–π—Å—Ç–≤:

>[!props] –°–≤–æ–π—Å—Ç–≤–∞
>1. $f=\Theta(g)$ —Ç–æ–≥–¥–∞ –∏ —Ç–æ–ª—å–∫–æ —Ç–æ–≥–¥–∞, –∫–æ–≥–¥–∞ $f=O(g)$ –∏ $f=\Omega(g)$.
>2. $f=O(g)$ —Ç–æ–≥–¥–∞ –∏ —Ç–æ–ª—å–∫–æ —Ç–æ–≥–¥–∞, –∫–æ–≥–¥–∞ $g=\Omega(f)$.
>3. $f=\Theta(g)$ —Ç–æ–≥–¥–∞ –∏ —Ç–æ–ª—å–∫–æ —Ç–æ–≥–¥–∞, –∫–æ–≥–¥–∞ $g=\Theta(f)$.
>{{% notice style="prove" expanded="true" %}}
–î–æ–∫–∞–∂–µ–º –¥–ª—è –ø—Ä–∏–º–µ—Ä–∞ –ø. 2.

–ü—É—Å—Ç—å $C>0$, —Ç–æ–≥–¥–∞ $|f(n)| \leqslant C \cdot|g(n)|$ —Ä–∞–≤–Ω–æ—Å–∏–ª—å–Ω–æ $|g(n)| \geqslant \frac{1}{C} \cdot|f(n)|$. 

–¢–∞–∫–∏–º –æ–±—Ä–∞–∑–æ–º, $f=O(g)$ —Å –∫–æ–Ω—Å—Ç–∞–Ω—Ç–æ–π $C$ —Ç–æ–≥–¥–∞ –∏ —Ç–æ–ª—å–∫–æ —Ç–æ–≥–¥–∞, –∫–æ–≥–¥–∞ $g=\Omega(f)$ —Å –∫–æ–Ω—Å—Ç–∞–Ω—Ç–æ–π $\frac{1}{C}$.
{{% /notice %}}

–°—É—â–µ—Å—Ç–≤—É—é—Ç —Ç–∞–∫–∂–µ –∞—Å–∏–º–ø—Ç–æ—Ç–∏—á–µ—Å–∫–∏–µ –∞–Ω–∞–ª–æ–≥–∏ $<$ –∏ $>: o(\cdot)$ –∏ $\omega(\cdot)$.
{{% notice style="def" title="–û–ø—Ä–µ–¥–µ–ª–µ–Ω–∏–µ o, $\omega$" %}}
–†–∞—Å—Å–º–æ—Ç—Ä–∏–º —Ñ—É–Ω–∫—Ü–∏–∏ $f, g: \mathbb{N} \rightarrow \mathbb{R}$.

- $f=o(g)$, –µ—Å–ª–∏ –¥–ª—è –ª—é–±–æ–≥–æ $C>0$ –Ω–∞–π–¥—ë—Ç—Å—è —Ç–∞–∫–æ–µ $N>0$, —á—Ç–æ –¥–ª—è –ª—é–±–æ–≥–æ $n>N$ : $|f(n)| \leqslant C \cdot|g(n)|$.
